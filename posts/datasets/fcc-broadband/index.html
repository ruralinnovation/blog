Overview

The FCC National Broadband Map provides detailed information about broadband availability across the United States. This dataset includes information on:

  • Broadband service provider coverage
  • Technology types (Fiber, Cable, DSL, etc.)
  • Upload and download speeds
  • Census block-level granularity

The National Broadband Map is the authoritative source for understanding broadband access gaps and measuring progress toward universal connectivity.

NoteWhy FCC Broadband Data is Essential
  • Location-level precision - Census block level data (finest geographic detail)
  • Technology detail - Fiber, cable, DSL, fixed wireless, satellite
  • Speed tiers - Upload and download speeds by provider
  • Biannual updates - More frequent than survey-based measures

Coverage

Geographic Coverage

  • Census block level
  • County aggregations
  • State aggregations
  • All 50 states

Temporal Coverage

  • Updated twice per year
  • Historical data available
  • Provider-reported service availability
  • Challenge process for data verification

Key Features

Geographic Granularity

Census block-level data allows for:

  • Precise gap analysis
  • Rural vs. urban comparisons
  • Infrastructure planning
  • Grant targeting (BEAD, ReConnect, etc.)

Technology Types

  • Wireline Technologies
  • Fixed Wireless
  • Satellite
  • Fiber - Fiber to the premises (FTTP/FTTH)
  • Cable - Hybrid fiber-coaxial (HFC)
  • DSL - Digital subscriber line
  • Copper - Legacy telephone lines
  • Licensed Fixed Wireless - Point-to-multipoint systems
  • Unlicensed Fixed Wireless - WiFi-based systems
  • 5G Home Internet - Cellular-based home broadband
  • Geostationary Satellite - Traditional satellite (high latency)
  • Low Earth Orbit (LEO) Satellite - Starlink, OneWeb, etc.

Speed Tiers

Upload and download speeds reported by providers:

  • 25/3 Mbps (legacy broadband definition)
  • 100/20 Mbps (BEAD threshold)
  • 1 Gbps+ (gigabit service)

Data Structure

Census Block Format:

  • One row per census block per provider per technology
  • Detailed service characteristics
  • Maximum advertised speeds

County Aggregated Format:

  • Unserved/underserved location counts
  • Technology availability percentages
  • Provider competition metrics

Methodology

Provider Reporting

Broadband service providers report coverage data to the FCC:

  • Service availability by census block
  • Maximum advertised speeds
  • Technology types deployed
  • Reported twice per year

Challenge Process

ImportantData Verification

The FCC uses a challenge process to verify provider-reported data:

  • State and local governments can challenge data
  • Service providers must respond to challenges
  • Data updated based on verified challenges
  • Improves accuracy over time

Fabric Data

The FCC uses “Fabric” data:

  • Comprehensive list of all broadband-serviceable locations (BSLs)
  • More accurate than census block aggregation
  • Enables location-level analysis

Strengths & Limitations

Strengths

  • Census block precision
  • Technology detail
  • Speed tier information
  • Challenge process improves accuracy
  • Biannual updates

Limitations

  • Provider self-reporting (verification ongoing)
  • Availability ≠ adoption
  • No pricing information
  • No quality/reliability metrics
  • Lag in challenge resolution

Important Notes

Availability vs. Adoption

WarningCritical Distinction

Availability: Broadband service is available at a location (from FCC map)

Adoption: Households subscribe to broadband service (from ACS)

FCC data measures availability, not actual subscriptions. Use ACS for adoption rates.

Speed Definitions

Different programs use different speed thresholds:

Program Download Upload Definition
Legacy FCC 25 Mbps 3 Mbps “Broadband”
BEAD 100 Mbps 20 Mbps “Underserved”
BEAD <25 Mbps <3 Mbps “Unserved”

Technology Considerations

Not all technologies are equal:

  • Fiber: Future-proof, symmetrical speeds, low latency
  • Cable: High download, lower upload, shared bandwidth
  • DSL: Distance-limited, slower speeds
  • Fixed Wireless: Weather-dependent, line-of-sight required
  • Satellite: High latency (except LEO), expensive
