FCC National Broadband Map

Comprehensive broadband availability and adoption data from the Federal Communications Commission

Updated: December 15, 2025
Broadband Infrastructure

Overview

The FCC National Broadband Map provides detailed information about broadband availability across the United States. This dataset includes information on:

  • Broadband service provider coverage
  • Technology types (Fiber, Cable, DSL, etc.)
  • Upload and download speeds
  • Census block-level granularity

Key Features

  • Geographic Coverage: Census block level data for all 50 states
  • Update Frequency: Updated twice per year
  • Data Format: Available in CSV and Parquet formats
  • API Access: Programmatic access via FCC API

Usage

This dataset is commonly used in:

  • Broadband equity analysis
  • Rural connectivity studies
  • Infrastructure planning
  • Policy research
